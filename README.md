## MyCareer

Another ReactJS app. Visit the [website](http://atefbb.github.io/cv-react-app).

## Technologies

-   React with hooks
-   TypeScript
-   Yarn

## Usage

-   `git clone` this repo,
-   Hit `yarn` to install dependencies,
-   create `.env.local` for local env variables (host/port),
-   Hit `yarn start` and you ready to GO!
